# Что планировала доработать 
В текущей версии не успела реализовать декоративные фигуры на фоне (геометрические элементы в углах экрана, как в макете). 
Изначально планировала добавить их через ::before / ::after или вёрстку SVG с абсолютным позиционированием, а также настроила бы их адаптивное масштабирование под разные разрешения.
